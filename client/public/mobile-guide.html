Using Lokisa as a Progressive Web App

1

Installing Lokisa on your device

Lokisa can be installed as a Progressive Web App, which allows you to use it like a native app on your device.

On Chrome (Android)

  1. Navigate to lokisa.replit.app
  2. You'll see an installation prompt at the bottom of the screen
  3. Tap "Install" to add Lokisa to your home screen

On Safari (iOS)

  1. Navigate to lokisa.replit.app
  2. Tap the Share button (box with arrow up)
  3. Scroll down and tap "Add to Home Screen"
  4. Tap "Add" in the upper right corner
2

Offline Functionality

Once installed, Lokisa can work offline for many functions:

  • Viewing previously loaded issues
  • Creating new reports (which will be submitted when you're back online)
  • Supporting existing issues (which will be registered when you're back online)

You'll see an offline indicator when your device doesn't have an internet connection.

Using Lokisa on iOS

1

Installing the App

You can install Lokisa on your iOS device in two ways:

Option 1: App Store (Coming Soon)

Lokisa will be available on the Apple App Store. Once available, you can search for "Lokisa" and install it directly.

Option 2: Add to Home Screen

Until the App Store version is available, you can install Lokisa as a web app:

  1. Open Safari and navigate to lokisa.replit.app
  2. Tap the Share button at the bottom of the screen (box with an arrow pointing up)
  3. Scroll down and tap "Add to Home Screen"
  4. Name the app "Lokisa" and tap "Add"
2

Using the Camera

When reporting an issue, tap the camera button to take a photo. On iOS, you'll be prompted to allow camera access the first time.

3

Location Services

Allow location access when prompted to accurately report and view issues near you.

Using Lokisa on Android

1

Installing the App

You can install Lokisa on your Android device in two ways:

Option 1: Google Play Store (Coming Soon)

Lokisa will be available on the Google Play Store. Once available, you can search for "Lokisa" and install it directly.

Option 2: Install from Web

Until the Play Store version is available, you can install Lokisa as a web app:

  1. Open Chrome and navigate to lokisa.replit.app
  2. Tap the prompt that appears at the bottom of the screen
  3. Or tap the menu (three dots) in the upper right and select "Install app"
2

Using the Camera

When reporting an issue, tap the camera button to take a photo. On Android, you'll be prompted to allow camera access the first time.

3

Location Services

Allow location access when prompted to accurately report and view issues near you.

General Tips

1

Reporting Issues

  • Be as specific as possible when selecting issue categories
  • Add a clear photo when possible - this helps officials identify the exact problem
  • Include notes about severity or safety concerns
  • Your reports are anonymous, but will be sent to relevant municipal officials
2

Supporting Issues

When you see an issue reported by another user that you also want fixed:

  • Tap the "Support" button on the issue
  • This increases the issue's priority and helps officials understand which problems affect the most people
3

Offline Usage

If you're in an area with poor connectivity:

  • Reports will be stored on your device and sent when connectivity returns
  • You'll see a notification when this happens
  • Previously loaded issues and maps will still be viewable offline
